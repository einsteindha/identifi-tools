# 재무 시뮬레이터 — 정원준 CFP

상담용 재무 시뮬레이터. 목적자금 적립, 주택구입 대출, 증여/양도세, 전세·월세 비교를 단일 HTML 파일로 제공합니다.

## 주요 기능

### 🎯 목적자금 시뮬레이터
- 목적자금 카드 다중 추가 (교육비, 여행, 자동차, 주택, 일반 등)
- 결정론적 계산 + 몬테카를로 시뮬레이션(3,000회) 병행
- 소득 성장률 반영 월 적립 자동 인상
- MC 성공률, 하위10%·중앙값·상위10% 경로 시각화
- 주택 구입 목적자금: 부동산 가격 상승률 설정 → 주택구입 탭 자동 연동
- 주택 갈아타기 카드: 현 주택 순자산 + 적립금 연동, 목표 주택 미래가·필요 대출·MC 하위10% 기준 대출 표시
- 한국부동산원 / KB부동산 실거래 참고 데이터 패널 (주택구입·갈아타기 카드 공통)

### 🔄 주택 갈아타기 분석
- 현 주택 순자산(시세 − 잔여대출) + 적립금 → 목표 주택 자기자금 자동 산출
- 갈아타기 시점·금액을 목적자금 탭 카드와 양방향 연동
- 신규 대출 LTV / DSR / 스트레스DSR / DTI 한도 산출
- 상환 방식별 비교, 연도별 차트, 부담 분석, PDF 출력 지원
- MC 하위10%(보수적) 시나리오 기준 추가 필요 대출 표시

### 🏠 주택구입 대출 분석

**대출 한도 계산**
- 실행 가능 대출 = MIN(LTV한도, DSR한도, 스트레스DSR한도) 자동 산출
- 자기자금 부족분 계산 — 필요 대출이 한도 초과 시 추가 필요 자기자금 표시
- 임차보증금(전세 세입자) LTV 한도 차감 반영 (금감원 규정)
- 기존 주담대·기타 대출 반영한 DSR 계산
- 소득 성장률 적용 미래 소득 기준 DSR (목적자금 탭 기간 연동)

**규제 지역별 LTV 한도**
- 토지거래허가구역·투기지역·투기과열지구: 무주택 40%
- 조정대상지역: 무주택 70% / 1주택 60% / 다주택 50%
- 일반 지역: 무주택·1주택 80% / 다주택 60%
- 토지거래허가구역(토허제) 거래 가능 여부 자동 판단 (2026.5.12 기준)

**상환 방식 비교**
- 원리금균등·원금균등·만기일시·혼합(거치) 4가지 초회 월납입금·총이자 비교
- 연도별 원금/이자 차트, 잔여원금 추이, 상환 스케줄 테이블
- 소득 대비 월 상환 부담률 분석

### 🎁 증여/양도 세금 비교
- 순수 증여 / 부담부 증여 / 일반 양도 3가지 방법 세금 자동 비교
- 증여세: 재산공제·세대생략 할증·혼인출산 추가공제·신고세액공제 반영
- 양도세: 1주택 비과세·고가주택 과세비율·장기보유특별공제·다주택 중과 적용
- 저가양도(상증세법 §35) / 고가양수 의제증여세 자동 판단
- 부당행위계산부인(소득세법 §101) 자동 판단 및 시가 기준 재산정

### 🏘️ 전세 vs 월세 비교
- 전세 자기부담 보증금 운용수익률 기준으로 유·불리 역산
- 연도별 비용 추이 차트 및 손익분기 수익률 시각화

## 사용 방법

별도 설치 없이 `index.html`을 브라우저에서 열면 바로 실행됩니다.

```
index.html 더블클릭 → 브라우저 실행
```

또는 GitHub Pages로 배포하여 URL로 접근할 수 있습니다.

## 기술 스택

- **순수 HTML/CSS/JavaScript** — 빌드 도구 없음, 단일 파일
- **Chart.js 4.4.1** — 시계열·막대·선 차트 (CDN)
- **Noto Sans KR / DM Mono** — Google Fonts (CDN)
